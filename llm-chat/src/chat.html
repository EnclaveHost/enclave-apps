    This deployment serves the same models over an OpenAI-compatible API — point any OpenAI SDK at:

    curl

    Streaming and non-streaming. Also GET /v1/models, GET /warmup?model=…. Model names match the picker. If the deployment config sets api_key, send it as Authorization: Bearer <key>. Requests run in the same enclave as this chat.
